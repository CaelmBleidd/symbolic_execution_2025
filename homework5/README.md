# Домашнее задание 5: Генерация тестов

## Цель
Реализовать компонент генерации тестов из результатов символьного анализа.

## Задания

### Задание 5.1: Реализация генератора тестов

В файле `test_generator.go` необходимо реализовать функцию, генерирующую тестовый файл для переданного файла.
В результате, функция должна возвращать путь до тестового файла (он должен располагаться рядом с переданным и название должно заканчиваться на `_test.go`), содержащего
сгенерированные тест-кейсы для всех функций, доступных в переданном на анализ файле. Тест-кейсы необходимо генерировать из 
результатов работы `analyser.Analyse`.

### Задание 5.2: Проверка работы генератора

Проверьте работу генератора тестов на примерах из **homework3** (`homework3/examples/test_functions.go`). 
Для этого используйте `coverage_checker.go` - он вызывает генератор тестов и выводит собранное тестовое покрытие. 

### Задание 5.3*: Добавление генерации в CI/CD

При желании, настройте CI/CD в репозитории таким образом, чтобы в нем был этап автоматического запуска `coverage_checker.go`
на файлах с примерами тестового кода.
