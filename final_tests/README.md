## Итоговые тесты

В данной директории содержатся примеры для приемочных тестов вашего проекта. 
Разработанный анализатор должен поддерживать анализ данных тестов. 

Предполагается запуск `coverage_checker.go` на всех тестовых файлах данной директории, поэтому, убедитесь,
что `test_generator.go` справляется с генерацией примеров для всех тестовых файлов. В результате, покрытие 
сгенерированных тестов должно стремиться к 100% при минимальном количестве тест-кейсов.
